Roblox Gemini AI Proxy

This server acts as a proxy between Roblox scripts and the Gemini AI API, helping to bypass Roblox's HTTP request limitations.

API Documentation

POST /api/gemini

Forwards requests to the Gemini API

Request Body:

{
  "prompt": "Your question or prompt for Gemini AI"
}

Response:

{
  "success": true,
  "response": "AI-generated response text"
}

Error Response:

{
  "success": false,
  "error": "Error message",
  "details": "Additional error details (if available)"
}

Test Your Setup

Visit /api/test to check if your server is running correctly.
